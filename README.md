# MIFARE_RC522_testing
MIFARE RC522 testing  

  
Arduino Nano pin# <-> RC522 pin#

3.3V - VCC  
GND - GND  
D9 - RST  
D10 - NSS  
D11 - MO  
D12 - MI  
D13 - SCK
